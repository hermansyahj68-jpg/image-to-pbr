# Image to PBR Texture Tool

Online tool to convert a single image into PBR texture maps.

🌐 Live Tool  
https://hermansyahj68-jpg.github.io/image-to-pbr/

## Features

- Upload image
- Generate PBR textures automatically
- Thumbnail preview
- Albedo / Color map
- Height / Bump map
- Normal map
- Download all maps as ZIP

## How to Use

1. Upload an image
2. Wait for the tool to generate textures
3. Download the PBR maps

## Generated Maps

- thumb.jpg
- color.jpg
- bmp.jpg
- normal.jpg
